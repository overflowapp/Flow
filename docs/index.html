The Node

All visible elements in a Flow file are Nodes. The following properties are shared between all Nodes, which helps us differentiate it amongst the rest.

Type Properties
Node id string

The unique identifier for Nodes

name string

A non-unique identifier for the node in the Document

type NODE_TYPE

A type, from a const enum, that declares what kind of node it is

Node Types

Each Node has its own type and has a different hierarcy in the structure. Nodes vary all the way from the Document Node, to the Hotspot Node. Nodes have several different uniquely associated properties as well as common properties.

Type Properties
DOCUMENT id string

The unique identifier for Nodes

name string

A non-unique identifier for the node in the Document

type NODE_TYPE

A type, from a const enum, that declares what kind of node it is

Other Types

There are several other types that ship with Flow to assist you in building a more consistent file structure.

Type Properties
Color

A RGBA color object

 r number

Red value, between 0 and 255

g number

Green value, between 0 and 255

b number

Blue value, between 0 and 255

a number

Alpha value, between 0 and 1
Point

An absolute (x, y) point

 x number

The absolute X coordinate in the space

y number

The absolute Y coordinate in the space
Size

A size object

 w number

The width, in pixels

h number

The height, in pixels
Connection

A connection between nodes

 nodeID string

The id of the Node that the connection ends up at

FileAsset

Name & path of an asset

 fileName string

The name of the file, with the extension (e.g image.png)

dirPath string

The relative path to the file
