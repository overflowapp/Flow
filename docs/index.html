The Node

All visible elements in a Flow file are Nodes. The following properties are shared between all Nodes, which helps us differentiate it amongst the rest.

Node Types

Each Node has its own type and has a different hierarcy in the structure. Nodes vary all the way from the Document Node, to the Hotspot Node. Nodes have several different uniquely associated properties as well as common properties.

Type Properties
Node id String

The unique identifier for Nodes.
